# Koimsurai GPS

這是一個 Android 應用程式，用於模擬 GPS 定位。使用者可以在地圖上選擇任何位置，並將其設定為裝置的全域 GPS 位置。

## ✨ 主要功能

*   **模擬定位**：將您的裝置 GPS 設定到世界任何角落。
*   **地圖介面**：使用 OpenStreetMap 提供互動式地圖。
*   **地址搜尋**：透過搜尋地址輕鬆找到目標位置。
*   **位置收藏**：儲存並快速選取您常用的地點。
*   **自訂標記**：現代化且清晰的地圖標記樣式。

## 🚀 設定與安裝

### 1. 安裝應用程式

*   **方法一 (使用 ADB - 推薦)**：
    ```bash
    adb install app/build/outputs/apk/debug/app-debug.apk
    ```

*   **方法二 (手動安裝)**：
    將 `app-debug.apk` 檔案傳輸到您的手機，然後點擊檔案進行安裝。（您可能需要授權「安裝未知來源的應用程式」）

### 2. 啟用開發人員選項

1.  前往「設定」 > 「關於手機」。
2.  連續點擊「版本號碼」七次，直到看見「您現在已成為開發人員」的提示。

### 3. 設定模擬位置應用程式

1.  前往「設定」 > 「系統」 > 「開發人員選項」。
2.  向下捲動並找到「選取模擬位置應用程式」。
3.  選擇「Koimsurai GPS」。

## 📖 如何使用

1.  **選擇位置**：直接點擊地圖，或在頂部搜尋框輸入地址來尋找位置。
2.  **儲存收藏 (可選)**：在搜尋到一個有名稱的地點後，點擊「Fav」按鈕即可將其加入收藏。您可以透過下拉選單快速選取已收藏的地點。
3.  **開始模擬**：點擊畫面下方的「Start Mock Location」按鈕。
4.  **停止模擬**：點擊「Stop Mock Location」按鈕。
